AngularJS
==========

AngularJS is a framework used mostly in single-page applications. It is a part
of MEAN software stack:

* MongoDB - a database
* Express.js - a back-end framework
* AngularJS - a front-end framework
* Node.js - a back-end platform

All of these four technologies are based on Javascript so they are enable to
develop in Plezuro. 

The examples below have origin from https://angularjs.org/ and later were
adapted to Plezuro.
